# Tip Calculator & Bill Splitter

A fast, responsive, single-screen tip calculator that updates live as you type. Built with vanilla web technologies for maximum performance and zero dependencies.

## How to Run Locally

Since this project uses vanilla HTML, CSS, and JS, there is no build step or package installation required.

**Steps:**

1. Clone or download this repository.
2. Open the folder on your computer.
3. Double-click the `index.html` file to open it in any modern web browser.
   _(Alternatively, you can open the folder in VS Code and use the "Live Server" extension for hot-reloading)._

## Features

- Live calculations (no "Calculate" button required)
- Graceful, flicker-free inline validation
- Fully responsive layout (mobile-first approach)
- Custom tip percentage input
- "Never underpay" rounding policy
